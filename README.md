# 五星连珠

漂亮的五子棋游戏

为了学习 canvas 并能够更好应用出来，我用 canvas 做了一个五子棋游戏做练习，界面一如既往的漂亮。
实现的功能有：人机交互（哦不，是人机大战）、锁定棋盘、悔棋、取消悔棋。

不稳定的传送门：https://Hounaoshao.github.io
